Preparar entorno:

```bash
npm i -g course-asynchrony
```

Crear un directorio de trabajo:

```bash
mkdir course-asynchrony
cd course-asynchrony
```

Obtener retos:

```bash
course-asynchrony
```

Verificar resultados:

```bash
course-asynchrony verify <FILE>
```
